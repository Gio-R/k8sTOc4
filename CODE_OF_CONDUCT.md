## Code of Conduct

We expect all participants to follow the Contributor Covenant Code of Conduct.

1. Be respectful and considerate.
2. Exercise empathy and assume good intentions.
3. Be mindful of the impact of your words and actions.
4. Seek to resolve conflicts respectfully and constructively.

If you experience or witness unacceptable behavior, please report it to the project maintainers using the issue tracker or contact methods provided in the repository.
